To run the project using exe 
____________________________________

1)Double click the 'mygui.exe' file..GUI will run within few seconds.

2)then draw any number from 0-9

3)click 'predict' button to get the prediction

4)click 'clear'button in order to remove the drawn digit.


to run without exe
_____________________
1) Go to the folder

2) open cmd from that folder path

3) Type and run python mygui.py
